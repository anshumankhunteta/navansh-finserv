# Navansh Finserv

A high-performance, SEO-optimized website for a small financial services firm. Built with a focus on trust, speed, and modern FinTech aesthetics.

## Tech Stack

- **Framework:** Next.js 16+ (App Router)
- **Styling:** Tailwind CSS
- **Components:** Shadcn UI
- **Icons:** Lucide React
- **Lead Gen:** Tally.so

## Getting Started

- **Clone:** `git clone https://github.com/anshumankhunteta/navansh-finserv`
- **Install:** npm install
- **​Dev View:** npm run dev
- **​Build:** npm run build
